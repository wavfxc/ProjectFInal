## 🌟 О Проекте (Чистый Сигма-Вайб)

Этот репозиторий — не просто набор папок. Это **абсолютная квинтэссенция стиля**, написанная на мощнейшем C# под соусом .NET MAUI. Мы взяли Steam, вывернули его наизнанку, прикрутили парсеры, кастомные конвертеры и создали ультимативный интерфейс для просмотра игр, от которого перехватывает дыхание!

Сюда заложена архитектура уровня ультра-босс: честный **MVVM**, сервисы, интерфейсы и медиаплеер, который заставит твой девайс потеть.

## 🔥 Фичи, Которые Разносят Кабину

* 🎮 **Steam Parser Service:** Парсит данные с такой скоростью, будто за ним бежит военкомат.
* 📺 **MediaElement Integration:** Крутит трейлеры игр прямо в приложении (если эмулятор не решит умереть).
* 🛠 **Custom Converters Pack:** Конвертируем цвета, строки и булевы значения быстрее, чем ты меняешь статус в ТГ.
* 📱 **Cross-Platform Power:** Теоретически работает везде, практически — заставляет Android-эмуляторы молить о пощаде.

## 🛠 Архитектурный Скриншот (Структура Настоящего Гигачада)

Каждая папочка лежит на своем месте, структура выверена до миллиметра:

```text
Project/
│
├── 📂 Converters/     # Чистим код от грязи, конвертируем как боги
├── 📂 Models/         # Мощнейшие дата-модели (SteamGame на базе)
├── 📂 Services/       # Сервисы и парсеры, которые держат всю логику
├── 📂 ViewModels/     # Связующий клей (GameDetailsViewModel детектед)
└── 📂 Views/          # Божественный XAML-интерфейс
