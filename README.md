# Project — трекер скидок Steam

Мобильное приложение на .NET MAUI, которое показывает актуальные скидки в Steam, умеет искать игры по всему каталогу, сортировать по цене и популярности, переключать регион (Россия / США / Украина) и показывать игры, недоступные для покупки в России.

## Что умеет

- Список игр со скидками — данные берутся прямо со страницы Steam
- Переключение региона с пересчётом цены в местной валюте
- Поиск по всему каталогу Steam, а не только по скидкам
- Сортировка: по популярности, по цене вверх/вниз
- Страница игры — описание, жанры, скриншоты, трейлер
- Отдельный блок с играми, заблокированными в России
- Анимации при нажатии на карточку и при открытии страницы игры
- Оформление в фирменных цветах Steam

## Как это работает

Приложение собрано по архитектуре MVVM: экран (XAML) ничего не знает про интернет, он просто смотрит на ViewModel через биндинги. ViewModel дёргает сервис, а сервис уже ходит в Steam.

Steam не даёт официальный API для списка скидок, поэтому страница поиска парсится как обычный HTML через библиотеку HtmlAgilityPack. А вот для деталей одной конкретной игры (описание, скриншоты, трейлер) используется официальный JSON API Steam — он надёжнее.

Список игр, заблокированных в России, собран из известных AppId (Hogwarts Legacy, GTA V и т.д.) и в реальном времени проверяется через Steam API — у Steam просто нет способа спросить "дай весь список заблокированных игр", можно только проверять по одной.

Все сервисы и ViewModel подключаются через встроенный DI-контейнер .NET MAUI — никаких `new` внутри классов, всё собирается автоматически в `MauiProgram.cs`.

## Технологии

- .NET 9 / .NET MAUI
- HtmlAgilityPack — парсинг HTML
- CommunityToolkit.Maui + MediaElement — воспроизведение трейлеров

## Запуск

1. Установи Visual Studio 2022 с нагрузкой **.NET Multi-platform App UI development**
2. Клонируй репозиторий и открой `Project.csproj`
3. `dotnet restore`
4. Выбери платформу (Android / Windows) и запусти

## Дисклеймер

Проект не связан с Valve или Steam, сделан в учебных целях.
